# Our Lab Web

## 项目介绍

    本项目是为了实验室网站而建立的。

## 目录结构描述

     |—— README.md                  //帮助文档
     |—— homepage.html              //主页html文件
     |—— research.html              //研究方向html文件
        |—— resaerch topic1.html    //方向一的html文件
     |—— member.html                //成员的html文件
     |—— dataset.html               //数据集的html文件
     |—— publications.html          //出版物的html文件
     |—— news.html                  //新闻的html文件
        |——news_0913_COMST          //新闻文件命名规则"news_date_topic"
     |—— css
        |——style.css                //存储css格式
